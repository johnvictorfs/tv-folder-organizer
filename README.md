## Development Setup

- Requirements

  - [pnpm](https://pnpm.io/)
  - [Node.js](https://nodejs.org/en/) (v18+)

- Install dependencies

  ```bash
  pnpm install
  ```

- Setup Database

  ```bash
  pnpm prisma db:push
  ```

- Run Dev Server

  ```bash
  pnpm dev
  ```

Prisma studio should now be running at http://localhost:5556 and the Next.js app at http://localhost:8093
